## Visual Design Notes

### Colors & Styling

**Row Highlighting**:
- Normal rows (New, Exact Duplicate): White background (`#ffffff`)
- Potential Duplicate rows: Yellow/warning background (`#fff3cd` - Bootstrap `.alert-warning` color)

**Icons**:
- Warning icon (âš ) appears in potential duplicate rows before the date column
- Checkmark (âœ“) in status pane for completed uploads
- Loading spinner (â³) during file processing

**Alert Box**:
- Uses Bootstrap `.alert .alert-warning` styling
- Warning icon on the left
- Dismissible (X button on right)

**Buttons**:
- Import: Primary button (blue)
- Delete All: Danger button (red)
- Browse: Secondary button (gray)

**Status Pane**:
- Uses Bootstrap `.alert .alert-info` styling
- Closable with X button on right
- Multiple status lines stack vertically

### Responsive Behavior (Deferred)
Desktop-first approach. Mobile/tablet optimization will be addressed after desktop implementation is complete.
